Theory

How LWR Works

LWR fits a separate weighted linear model for every query point x₀ using nearby training data.

01
🎯

Pick a Query Point x₀

To predict at x₀, a local weighted linear model is fitted using all training points — but not equally.

02
🔔

Compute Gaussian Kernel Weights

Each training point gets a weight based on its distance to x₀ using the Gaussian kernel:

w(i) = exp(−‖x⁽ⁱ⁾ − x₀‖² / 2τ²)
03
⚖️

Weighted Least Squares

Solve the weighted normal equations to find local parameters θ:

θ = (XᵀWX)⁻¹ XᵀWy
04
📍

Local Prediction

Use θ to predict only at x₀ — then repeat for every query point in the test set.

ŷ(x₀) = [1, x₀]ᵀ · θ

Source Code

Python Implementation

Full implementation using only NumPy and Matplotlib — no sklearn required.

lwr.py 
import numpy as np
import matplotlib.pyplot as plt

# ── Reproducible random seed ──────────────────────────
np.random.seed(0)

# ── Generate training data: sin(x) + Gaussian noise ──
X = np.linspace(-3, 3, 100).reshape(-1, 1)
y = np.sin(X).ravel() + np.random.normal(0, 0.1, 100)

# ── Gaussian kernel function ──────────────────────────
def kernel(x0, X, t):
    """Weight each point by distance to query x0."""
    return np.exp(-np.sum((X - x0)**2, axis=1) / (2 * t**2))

# ── LWR prediction at a single point x0 ──────────────
def predict(x0, t=0.5):
    """Locally weighted linear regression prediction."""
    W     = np.diag(kernel(x0, X, t))          # weight matrix
    X1    = np.c_[np.ones(len(X)), X]           # add bias column
    theta = np.linalg.pinv(X1.T @ W @ X1) @ X1.T @ W @ y
    return np.r_[1, x0] @ theta

# ── Generate predictions over dense grid ─────────────
Xt = np.linspace(-3, 3, 300).reshape(-1, 1)
yp = np.array([predict(x) for x in Xt])

# ── Plot ───────────────────────────────────────────────
plt.figure(figsize=(10, 6))
plt.scatter(X, y, color='#f472b6', s=40, alpha=0.7, label='Training data')
plt.plot(Xt, yp, color='#fb923c', linewidth=2.5, label='LWR fit (τ=0.5)')
plt.xlabel('x'); plt.ylabel('y')
plt.title('Locally Weighted Regression')
plt.legend(); plt.grid(alpha=0.3)
plt.tight_layout(); plt.show()
